
Problem: Largest Number in Array

You are given an array of integers. Write a function to find and return the largest number present in the array.

Function Signature:
function findLargestNumber(numsArr) {
    // Your code here
}

Output:
The largest number present in the input array.

Test Case-1:
findLargestNumber([3, 7, 2, 8, 5]);
// Expected Output: 8
// Explanation: The largest number in the array is 8.

Test Case-2:
findLargestNumber([-10, -5, -8, -3]);
// Expected Output: -3
// Explanation: The largest number in the array is -3.

Test Case-3:
findLargestNumber([1, 1, 1, 1]);
// Expected Output: 1
// Explanation: All numbers in the array are the same, so the largest number is 1.



NOTE:

The input array may contain both positive and negative integers.

You are not required to handle cases where the array is empty.
